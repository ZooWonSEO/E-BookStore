## eBookstore

# 프로젝트의 목적
습득한 HTML, javascript을 이용해 직접 웹을 설계하고 개발해보기

# 기능설계
1. 관리자가 상품등록 및 관리
2. 관리자가 고객관리
3. 게시판을 통해 물건에 대한 정보 올리기
4. 고객이 물품을 구매하고 구매목록 확인

# 개발환경
- 사용언어 : javascript, html, java
- jdk14.0.1
- local server : tomcat 8.5 version
- DB : firebase
- 디자인 : bootstrap
